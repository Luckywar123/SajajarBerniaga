SETUP DATABASE
Supabase + PostgreSQL
Panduan lengkap menghubungkan sistem MBG ke Supabase sebagai backend database. Ikuti langkah-langkah berikut secara berurutan.
01
BUAT AKUN SUPABASE
Kunjungi supabase.com dan buat akun gratis. Klik "New Project", beri nama mbg-system, pilih region Southeast Asia (Singapore) untuk latensi terbaik dari Indonesia. Catat Project URL dan anon public key dari Settings → API.
02
BUAT TABEL DATABASE
Buka SQL Editor di Supabase, jalankan query berikut:
-- Tabel buah/produk CREATE TABLE fruits ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, name VARCHAR(100) NOT NULL, latin_name VARCHAR(150), category VARCHAR(50), price_per_kg DECIMAL(10,2), stock_kg DECIMAL(10,2) DEFAULT 0, min_stock_kg DECIMAL(10,2) DEFAULT 50, emoji VARCHAR(10), created_at TIMESTAMPTZ DEFAULT NOW() ); -- Tabel batch stok (FIFO) CREATE TABLE stock_batches ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, fruit_id UUID REFERENCES fruits(id), batch_code VARCHAR(50) UNIQUE NOT NULL, entry_date DATE NOT NULL, expiry_date DATE NOT NULL, qty_kg DECIMAL(10,2) NOT NULL, qty_remaining DECIMAL(10,2), location VARCHAR(100), status VARCHAR(20) DEFAULT 'active', created_at TIMESTAMPTZ DEFAULT NOW() ); -- Tabel pesanan CREATE TABLE orders ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, order_number VARCHAR(20) UNIQUE NOT NULL, customer_name VARCHAR(200), destination VARCHAR(300), destination_type VARCHAR(50), -- sekolah, posyandu, dll items JSONB, total_weight_kg DECIMAL(10,2), status VARCHAR(30) DEFAULT 'pending', proof_photo_url TEXT, notes TEXT, created_at TIMESTAMPTZ DEFAULT NOW(), completed_at TIMESTAMPTZ ); -- Tabel nutrisi per buah CREATE TABLE nutrition_info ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, fruit_id UUID REFERENCES fruits(id) UNIQUE, calories_kcal DECIMAL(8,2), carbs_g DECIMAL(8,2), protein_g DECIMAL(8,2), fat_g DECIMAL(8,2), fiber_g DECIMAL(8,2), vitamin_c_mg DECIMAL(8,2), potassium_mg DECIMAL(8,2) ); -- Tabel staff/pengguna CREATE TABLE staff ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, username VARCHAR(50) UNIQUE NOT NULL, full_name VARCHAR(200), role VARCHAR(30), -- admin, kurir, gudang is_active BOOLEAN DEFAULT true, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Tabel tracking pengiriman CREATE TABLE deliveries ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, order_id UUID REFERENCES orders(id), driver_id UUID REFERENCES staff(id), lat DECIMAL(10,6), lng DECIMAL(10,6), status VARCHAR(30), updated_at TIMESTAMPTZ DEFAULT NOW() ); -- Tabel laporan bulanan (view) CREATE VIEW monthly_report AS SELECT DATE_TRUNC('month', created_at) AS month, COUNT(*) AS total_orders, SUM(total_weight_kg) AS total_weight_kg, COUNT(CASE WHEN status = 'completed' THEN 1 END) AS completed, AVG(total_weight_kg) AS avg_weight_per_order FROM orders GROUP BY DATE_TRUNC('month', created_at) ORDER BY month DESC;
03
AKTIFKAN ROW LEVEL SECURITY
Aktifkan RLS untuk keamanan data. Hanya staff terautentikasi yang dapat mengubah data.
-- Aktifkan RLS ALTER TABLE fruits ENABLE ROW LEVEL SECURITY; ALTER TABLE stock_batches ENABLE ROW LEVEL SECURITY; ALTER TABLE orders ENABLE ROW LEVEL SECURITY; -- Policy: publik bisa baca katalog buah CREATE POLICY "Public can read fruits" ON fruits FOR SELECT USING (true); -- Policy: hanya authenticated bisa ubah stok CREATE POLICY "Staff can manage stock" ON stock_batches FOR ALL USING (auth.role() = 'authenticated'); -- Policy: authenticated bisa manage orders CREATE POLICY "Staff can manage orders" ON orders FOR ALL USING (auth.role() = 'authenticated');
04
SETUP STORAGE UNTUK FOTO
Buat bucket di Supabase Storage untuk foto bukti penerimaan.
-- Di Supabase Dashboard → Storage → New Bucket -- Nama bucket: "proof-photos" -- Public: false (private) -- Policy upload foto (hanya staff) CREATE POLICY "Staff can upload proof photos" ON storage.objects FOR INSERT WITH CHECK ( bucket_id = 'proof-photos' AND auth.role() = 'authenticated' ); -- Policy read foto CREATE POLICY "Staff can view proof photos" ON storage.objects FOR SELECT USING ( bucket_id = 'proof-photos' AND auth.role() = 'authenticated' );
05
HUBUNGKAN KE APLIKASI
Install Supabase JS client dan konfigurasi koneksi.
